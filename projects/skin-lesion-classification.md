# Skin Lesion Classification - Deep Learning Research

## Overview
Built an ensemble deep learning model on the HAM10000 dataset combining Swin-B Transformer, ConvNeXt-B, and EfficientNetV2-L using PyTorch and Hugging Face Transformers; achieved multi-class medical image classification across 7 skin lesion categories.

## Stack
- PyTorch
- Hugging Face Transformers
- Python

## Key Engineering Decisions
- Swin-B, ConvNeXt-B, and EfficientNetV2-L ensemble design
- Conformal prediction for uncertainty quantification
- Grad-CAM visual explanations (Explainable AI)
